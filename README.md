## Podrozdział

Fragment kodu:

```csharp
int x = 1;
Console.Write(x);
```

A teraz kod wtrącony w tekst: `int y = 100;`. Tez zapis deklaruje zmienną `y` typu całkowitego i przypisuje jej wartość `100`.
